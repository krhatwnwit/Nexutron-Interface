==================================================
Nexutron_Interface  - 端末風テキスト表示ツール
==================================================

■ 説明：
本ツールは、事前に用意したテキストファイルを
「ハッキング風」「SFコンソール風」のアニメーション表示で再生するデモ用ツールです。

色の指定、処理中風スピナー表示、テキストブロックごとの出力など、
動画制作やイベント演出にご利用いただけます。

--------------------------------------------------
■ 操作方法：

① 本アプリを起動します（Nexutron_Interface.exe）

② 最初にテキストファイルを選択するダイアログが表示されます。
　 → サンプル： sample_commands.txt を選んでください。

③ 表示サイズやフォントサイズを指定してください。
　 → 例: 16:9（デフォルト）／28pt

④ 再生 ▶ ボタンを押すとアニメーションが開始されます。

⑤ ファイルを変えたいときは 📂 ボタンから再選択できます。
　 表示サイズを変更したいときは ⚙ ボタンから可能です。

--------------------------------------------------
■ テキストファイルの書き方（ルール）

● `[GREEN]` や `[RED]` などのカラータグを行頭に入れると、以降のブロックに色が付きます。
● 空行を入れることで「次のブロック」となります。
● テキストは行単位でそのまま出力されます。

例：

[GREEN]
>> SYSTEM ONLINE
>> AUTH SUCCESS

[YELLOW]
>> LOADING MODULE...
>> WARNING: UNSTABLE CONNECTION

[RED]
>> ERROR: ACCESS DENIED

--------------------------------------------------
■ 注意：

・本アプリケーションは演出目的のソフトウェアであり、実際のシステム操作やネットワーク動作は行いません。
・内容はすべて事前に用意したテキストファイルに基づきます。
